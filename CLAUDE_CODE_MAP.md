## Qué NO significa “tener control total”

Aunque Claude Code esté operativo, eso no significa control absoluto de todo.
Todavía hay límites por:

- permisos del sistema operativo
- logins web/sesiones del navegador
- accesos a cuentas de terceros
- políticas del entorno / sandbox
- restricciones de herramientas externas
- cambios en UI de sitios web

## Traducción práctica

### Sí tengo control fuerte sobre:
- tareas de código
- scripts
- automatización técnica
- revisión/edición de archivos
- flujos complejos dentro del workspace

### No tengo control garantizado sobre:
- cualquier web privada sin sesión válida
- integraciones no configuradas
- acciones que dependan de permisos externos no otorgados
- automatizaciones frágiles de UI si no se preparan bien
